# Workshop paper -- QueryConditionedF (NeurIPS 2026 workshop, sept)

## Build

```bash
cd paper/workshop
pdflatex main.tex && bibtex main && pdflatex main.tex && pdflatex main.tex
```

Outputs `main.pdf`.

## Files

- `main.tex` -- root document, loads sections via `\input`.
- `section_*.tex` -- one file per section.
- `references.bib` -- BibTeX bibliography.
- `figures/` -- PDF figures generated by `python -m kiki_flow_core.track3_deploy.eval.kl_species` after T22/T23.
